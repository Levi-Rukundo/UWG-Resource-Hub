## Project Description
The UWG Campus Resource Hub is a student-created website designed to help University of West Georgia
students easily find important campus resources in one place. The website organizes resources into 
categories such as Academic Support, Wellness & Health, Technology Help, Career Services, and Basic Needs.

This project was developed as part of a web development course and demonstrates the use of HTML and CSS 
to create a structured, user-friendly, and responsive website.

## Usage Instructions
- Use the navigation bar at the top of the page to move between different sections of the website.
- Click on category cards on the homepage to access specific resource pages.
- Browse information on each page to learn about available campus services.
- Use the Contact page to submit questions or feedback through the form.

## Disclaimer
This website is a student project created for an academic course and is not an official website of 
the University of West Georgia.
