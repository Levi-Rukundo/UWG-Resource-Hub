About This Website

The UWG Campus Resource Hub is a student-created project designed to organize campus support resources in one simple and easy-to-use place.

Project Purpose

The purpose of this website is to help students quickly locate important campus resources related to academics, wellness, technology, career development, and basic needs.

Target Audience

This website is designed for college students, especially first-year students, transfer students, and others who may need guidance finding support services.

Disclaimer

This website is a student project created for an academic course and is not an official website of the University of West Georgia.

Future Improvements

Future versions of this website could include direct links to campus offices, interactive search tools, and more detailed resource information.
